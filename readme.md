# 🎮 Tic Tac Toe

A simple and responsive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

* Two-player gameplay (X vs O)
* Winner detection
* Draw detection
* Restart game functionality
* Responsive design for mobile and desktop
* Modern UI with gradient background and hover effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Play

1. Player **O** starts the game.
2. Players take turns placing their marks (O and X).
3. The first player to get three marks in a row wins.
4. If all nine cells are filled and no player wins, the game ends in a draw.
5. Click **Restart Game** to start a new match.

## 🏆 Winning Conditions

A player wins by occupying any of the following combinations:

* Horizontal rows
* Vertical columns
* Diagonal lines

## 📱 Responsive Design

The game adapts to different screen sizes, including:

* Mobile Phones
* Tablets
* Laptops
* Desktops

## 🔄 Restart Functionality

The restart button:

* Clears the board
* Resets player turns
* Resets game state
* Removes previous winner/draw messages
